# Mini-Rt
Very simple ray tracing library

![raytracing result](doc/raytracing.jpg)

## Features
- Objects: spheres
- Lights: point lights
- Phong shading
- Reflection and refraction (transparency)
- Saving result in JPEG
- Loading scene from file
